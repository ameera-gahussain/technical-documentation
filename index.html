What is it?

Javascript is a programming language created for the web. Alongside HTML and CSS, Javascript can make your websites come to life.

Today, besides using it for cool functions on websites, Javascript can also be used for programming a robot (with an Arduino), writing a game script in Unity and building code editors (and a whole lot more, so I’m told.)

Data Types and Variables

Data & data types are the building blocks of any programming language because they help us organise information and determine how our programs will run.

The following are primitive data types in Javascript

  • Numbers
  • Strings
  • Booleans
  • Undefined
  • Null

Variables on the other hand, are containers for storing data.

What is NaN?

stands for "Not-A-Number" and it's often returned indicating an error with number operations. For instance, if you wrote some code that performed a math calculation, and the calculation failed to produce a valid number, NaN might be returned.

// calculating the square root of a negative number will return NaN Math.sqrt(-10) // trying to divide a string by 5 will return NaN "hello"/5
Conditionals

Conditional statements control behaviour in JavaScript and determine whether or not pieces of code can run.

There are multiple different types of conditionals in JavaScript including:

  • “If” statements: where if a condition is true it is used to specify execution for a block of code.
  • “Else” statements: where if the same condition is false it specifies the execution for a block of code.
  • “Else if” statements: this specifies a new test if the first condition is false.
Loops

Loops are for executing blocks of code over and over again. Loops can execute a block of code as long as a specified condition is true.

There are 3 types

  • While Loop
  • For Loop
  • Nested Loops

A note on incrementing and decrementing:

x++ or ++x // same as x = x + 1 x-- or --x // same as x = x - 1 x += 3 // same as x = x + 3 x -= 6 // same as x = x - 6 x *= 2 // same as x = x * 2 x /= 5 // same as x = x / 5
Functions

Functions package up code so you can easily use (and reuse) a block of code. A JavaScript function is executed when "something" invokes it (calls it).

Arrays

An array  is useful because it stores multiple values into a single, organized data structure. You can define a new array by listing values separated with commas between square brackets [].

const cars = ["Saab", "Volvo", "BMW"];

In arrays, you can store strings, numbers, booleans and even another array (i.e.***nested arrays***)! Nested arrays can be particularly hard to read, so it's common to write them on one line, using a newline after each comma:

var arraysInArrays = [ [1, 2, 3], ["Julia", "James"], [true, false, true, false] ];
Objects

JavaScript objects are a data structure in JavaScript that lets you store data about a particular thing, and helps you keep track of that data by using “key”.

var sister = { name: "Sarah", age: 23, parents: [ "alice", "andy" ], siblings: ["julia"], favoriteColor: "purple", pets: true };
